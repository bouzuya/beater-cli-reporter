# beater-reporter

default [beater][bouzuya/beater] reporter.

[bouzuya/beater]: https://github.com/bouzuya/beater

## Installation

### for `beater@0.3.0`

```
$ npm install beater-reporter@0.1.0
```

### for `beater@0.4.x`

```
$ npm install beater-reporter
```

## Usage

```js
$ beater --reporter beater-reporter
```

or

`package.json`

```js
{
  "beater": {
    "reporter": "beater-reporter"
  }
}
```
